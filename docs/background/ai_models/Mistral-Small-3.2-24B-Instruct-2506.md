# Mistral-Small-3.2-24B-Instruct-2506

Note the `2506` refers to the 25 June 2025, where the model were released

Generative model (instruct fashion) with vision capabilities.

- see [huggingface model card](https://huggingface.co/mistralai/Mistral-Small-3.2-24B-Instruct-2506)
- alt see [Mistral blog post](https://mistral.ai/news/mistral-small-3) but note that the benchmarks are for the 
  25 January 2025 version

From [EuroEval danish leaderboard](https://euroeval.com/leaderboards/Monolingual/danish/) it gets 
- an overall rank of 1.48 (the closer to one the better) 
  - a placement as number 15 on the leaderboard (as of 4 February 2026)
- European values alignment score: Not available

Results from [AI-arenaen](https://ai-arenaen.dk/) is not yet available.

Using the [Linux Foundation](https://lfaidata.foundation/) [Model Openness framework (MOF)](https://isitopen.ai/)
Mistral Small 3 is deemed to be a class 3 open model (an "open weights model") (class 1 is best/most open). 
[Alternative take/additional info on model openess from Mozilla](https://assets.mofoprod.net/network/documents/Towards_a_Framework_for_Openness_in_Foundation_Models.pdf).
[Another alternative from the OpenSource Initiative](https://opensource.org/ai)
